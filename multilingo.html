MultiLingo

Project Context

MultiLingo strives to break the language barriers that we have today. Using the industry's most trusted and pioneer model, Chat GPT 3.5, we made a application that translates from language to language and teaches the user simple phrases that aid with travelling.

Find Out More GitHub

Reflection

From this project, I learned not only how to tap into Large Language Models (LLMs) but also how to harness their strengths in language processing. During the project, my group and I encountered various issues, such as the LLM generating gibberish or incorrect outputs. However, we persevered, continually refining our code and prompts to achieve the desired results.

Key Learning Points

- How to tap into AI models with Python.

- How to use Streamlit to make an interface.

- How to write a bash file.
